Mathematical functions
    Function        | Description                               | Example
    --------------------------------------------------------------------------------------------
    CEILING(x)      | Returns the smallest integer greater or   | CEILING(4.2) = 5
                    | equal to x                                | CEILING(-5.8) = -5
    --------------------------------------------------------------------------------------------
    ROUND(x, k = 0) | Round the number x up to k leading digits | ROUND(4.361) = 4
                    | after the floating point                  | ROUND(5.86592,1) = 5.9
    --------------------------------------------------------------------------------------------
    FLOOR(x)        | Returns the greatest integer smaller or   | FLOOR(4.2) = 4
                    | equal to x                                | FLOOR(-5.8) = -6
    --------------------------------------------------------------------------------------------
    POWER(x, y)     | Return x to the power y                   | POWER(3,4) = 81.0
    --------------------------------------------------------------------------------------------
    SQRT(x)         | Return square root of x                   | SQRT(4) = 2.0
                    |                                           | SQRT(2) = 1.41...
    --------------------------------------------------------------------------------------------
    DEGREES(x)      | Convert x from radians to degrees         | DEGREES(3) = 171.8...
    --------------------------------------------------------------------------------------------
    RADIANS(x)      | Convert x from degrees to radians         | RADIANS(180) = 3.14...
    --------------------------------------------------------------------------------------------
    ABS(x)          | Return absolute value of x                | ABS(-1) = 1
    --------------------------------------------------------------------------------------------
    PI()            | Return pi                                 | PI() = 3.1415926...
    --------------------------------------------------------------------------------------------
    RAND()          | Return random number from the range       | RAND() = 0.1324124
                    | [0, 1]                                    |
